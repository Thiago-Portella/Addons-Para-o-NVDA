Usage

This add-on serves to record in a text document all speeches of NVDA while the add-on is active. After recording stops, a text document will be saved in the NVDA user configuration directory, and the recorded text is shown in a dialog. In this dialog you can: * Open your NVDA user configuration directory; * Copy the recorded text to clipboard; * Close the dialog.

Note: What is spoken in SayAll is not recorded.

To activate/deactivate the recording Press ALT + NumPad Plus. This command can be modified in the "Input gestures" dialog in the "Miscellaneous" category.
